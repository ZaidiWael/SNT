Programmation Python


La programmation en Python sert à donner des instructions à un ordinateur pour qu’il réalise des tâches. C’est comme donner une recette très précise à une machine.

Exemple :

print("Bonjour tout le monde !")

Ce petit programme affiche « Bonjour tout le monde ! » à l’écran.

Python est utilisé pour :

Créer des sites web.
Programmer des jeux vidéo.
Analyser des données scientifiques.

Fun Fact : Python est nommé d’après les Monty Python, un groupe de comédiens britanniques !

Données Structurées



Les données structurées sont des informations bien organisées pour être facilement utilisées.

Exemples :

Un tableau Excel avec des notes d’élèves.

Une base de données d’une bibliothèque (titre, auteur, date).

Pourquoi c'est utile ?
Si les données sont bien rangées, on peut rapidement trouver une information ou faire des calculs automatiques.

Par exemple, pour savoir qui a eu la meilleure note dans la classe !

Les Réseaux Sociaux


Les réseaux sociaux permettent aux gens de se connecter, de partager des photos, des vidéos et de discuter.

Exemples :

Instagram : partage de photos.

TikTok : création de courtes vidéos.

Snapchat : envoi de messages éphémères.

Attention ! Ces réseaux utilisent nos données personnelles pour :

Nous proposer du contenu.
Nous afficher de la publicité ciblée.

Conseil : toujours réfléchir avant de poster !

Informatique Embarquée & Objets Connectés


L’informatique embarquée, c’est quand un ordinateur est intégré dans un objet du quotidien.

Exemples :

Une montre connectée qui compte tes pas.

Une voiture avec GPS intégré.

Une caméra de sécurité connectée à ton téléphone.

Objets connectés : Ils peuvent communiquer entre eux via Internet (par exemple : Alexa ou Google Home).

Photographie Numérique



La photographie numérique transforme la lumière en données numériques.

Exemples de formats d’images :

JPEG : utilisé pour les photos classiques.
PNG : idéal pour les images avec fond transparent.

Processus :

La lumière entre dans l’appareil.

Un capteur transforme la lumière en pixels (petits carrés de couleurs).

L’image est sauvegardée sur une carte mémoire.

Exemple amusant : Une photo de 12 mégapixels contient environ 12 millions de petits carrés !

Localisation


La localisation permet de connaître ta position sur Terre grâce aux satellites.

Exemple :

Quand tu ouvres Google Maps, ton téléphone utilise le GPS pour savoir où tu es.

Comment ça marche ?

Ton téléphone communique avec plusieurs satellites.

Grâce au temps que le signal met à revenir, il calcule ta position.

Utilisation :

Trouver une adresse.

Partager ta position en temps réel.

Cartographie


La cartographie est l’art de créer des cartes géographiques.

Exemples de cartes modernes :

Google Maps.

Plans de randonnées numériques.

Aujourd'hui, on utilise des outils numériques pour :

Mesurer des distances.

Ajouter des informations en direct (trafic, restaurants, etc.).

Anecdote : Avant, les cartes étaient dessinées à la main et pouvaient contenir beaucoup d’erreurs !

Internet & Web


Internet est le réseau qui relie les ordinateurs entre eux. Le Web est un service d’Internet qui permet d’accéder à des sites.

Exemples de services Internet :

Les emails (Gmail).

Les jeux en ligne.

Le Web (Google, YouTube).

Fonctionnement d'une page web :

Tu tapes une adresse (ex : www.youtube.com).

Ton navigateur demande la page au serveur.

Le serveur envoie la page qui s’affiche.

Différence simple :

Internet = autoroute.

Web = voitures qui roulent dessus !
